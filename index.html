    Part 1 - Get your VM working (75%)

    1. Download and install VirtualBox on your local computer or laptop (https://www.virtualbox.org/wiki/Downloads)
    2. Download Ubuntu Server (http://www.ubuntu.com/download/server). If you install the Desktop version, I send it back :)
    3. Create a new virtual machine and install Ubuntu Server.

    Part 2 - Install Apache (25%)

    1. Use apt-get to install Apache, which is a basic web server
    2. Set up port-forwarding in Virtual Box, so you can see your new web-server by going to http://127.0.0.1:80/

    See the Take Home Exam section on the syllabus for instructions on how to turn in this assignment.
